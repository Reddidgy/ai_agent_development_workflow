## Contribution Guidelines

1. Create your branch from the `main` branch.
2. Follow the project structure and naming conventions.
3. Make your changes and commit them with clear, descriptive messages.
4. Push your branch to the remote repository.
5. Open a pull request and provide a detailed description of your changes in the following format:
   - **Title**: Short, clear, and concise.
   - **Description**: Detailed explanation of the changes made, the reason for the changes, and any relevant context.

## Code of Conduct

We are committed to providing a welcoming and inclusive environment for all contributors.
Please adhere to the following code of conduct:
- Be respectful and considerate in all interactions.
- Avoid any form of harassment or discrimination.
- Report any issues or concerns to the project maintainers.
- Follow the project's guidelines and standards for contributions.
- Collaborate and communicate effectively with other contributors.
- Strive for constructive feedback and continuous improvement in the project.
